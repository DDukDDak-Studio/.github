## 🗂 Repository Guide

이 오가니제이션은 아래 규칙으로 레포지토리를 관리합니다.

- `game-*` : 실제 게임 프로젝트
- `proto-*` : 시스템 / 기믹 실험용 프로토타입
- `tools-*` : 개발 편의 도구
- `.github` : 조직 공통 문서 및 가이드
